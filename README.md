# Flight Search Automation Framework

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Page Object Model (POM)
- Git

## Features
- Flight search automation on Yatra
- Cross-browser execution (Chrome, Opera & Edge)
- Dynamic travel date selection
- Explicit waits
- Assertions

## Run Tests

pytest -s --browser=chrome

pytest -s --browser=edge

pytest -s --browser=opera
